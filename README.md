## 📖 Описание

**Расписной Покер** — это классическая карточная игра в онлайн-формате. Играйте с друзьями в кампании из нескольких режимов, каждый со своими уникальными правилами!

### 🎯 Особенности

- ✅ **5 игровых режимов** с разными правилами
- ✅ **2-4 игрока** в одной комнате
- ✅ **Умный джокер** (6♠) с выбором силы и масти
- ✅ **Авто-реконнект** при обрыве связи
- ✅ **Адаптивный дизайн** для ПК и мобильных
- ✅ **Система кампании** с переходом между режимами
- ✅ **Тест-режим** для отладки и тренировки

## 🃏 Игровые режимы

| Режим | Козырь | Торговля | Раундов | Очки |
|-------|--------|----------|---------|------|
| 🎯 **Классика** | ✅ Случайный | ✅ Да | 13-36* | Угадал заявку |
| 😈 **Мизер** | ✅ Случайный | ❌ Авто 0 | = игроков | 0 взяток = +20 |
| 🃏 **Бескозырка** | ❌ Нет | ✅ Да | = игроков | Угадал заявку |
| 👁️ **Слепая** | ✅ Случайный | ✅ Да | = игроков | Угадал заявку ×2 |
| 🔥 **Хапки** | ✅ Случайный | ❌ Авто | = игроков | +20 за взятку ×2 |

*В Классике количество раундов зависит от числа игроков (рост → плато → спад)

## 📜 Правила игры

### 🎴 Базовые правила

1. **Колода:** 36 карт (6-10, J, Q, K, A)
2. **Джокер:** 6♠ — универсальная карта
3. **Цель:** Точно угадать количество взяток (кроме Мизера и Хапок)

### 🃏 Джокер (6♠)
┌─────────────────────────────────────────┐
│  🃏 ДЖОКЕР — ПОЛНЫЙ ФУНКЦИОНАЛ          │
├─────────────────────────────────────────┤
│  Первый ход во взятке:                  │
│  ├─ Выбор масти (♠♥♦♣)                  │
│  ├─ Выбор силы (⬆️ Старшая / ⬇️ Младшая) │
│  └─ Масть хода = выбранная масть        │
│                                         │
│  Ответ на ход:                          │
│  ├─ Только выбор силы                   │
│  ├─ Старшая → выигрывает взятку         │
│  └─ Младшая → проигрывает любой карте   │
│                                         │
│  Логика победы:                         │
│  ├─ Joker HIGH > Любая карта            │
│  ├─ Любая масть/козырь > Joker LOW      │
│  └─ Joker LOW < Joker HIGH              │
└─────────────────────────────────────────┘


### 📊 Подсчёт очков

| Режим | Формула |
|-------|---------|
| 🎯 Классика | Угадал: `10 × bid`, Не угадал: `-10 × (bid - tricks)` |
| 😈 Мизер | 0 взяток: `+20`, Иначе: `-10 × tricks` |
| 🃏 Бескозырка | Как Классика |
| 👁️ Слепая | Как Классика ×2 |
| 🔥 Хапки | `+20 × tricks` ×2 |

## 🚀 Установка

### Требования

- Node.js 18+
- npm 8+

### Шаги установки


```bash
# 1. Клонируйте репозиторий
git clone <repository-url>
cd raspisnoy-poker

# 2. Установите зависимости
npm install

# 3. Запустите сервер
npm start

# 4. Откройте в браузере
http://localhost:3000
```
### 🎮 Использование
Создание комнаты
1.	Введите ваше имя
2.	Нажмите “🏠 Создать комнату”
3.	Скопируйте код комнаты (6 символов)
4.	Отправьте код друзьям
Вход в комнату
1.	Введите ваше имя
2.	Введите код комнаты
3.	Нажмите “🚪 Войти”
Начало игры
•	Минимум 2 игрока для старта
•	Игрок №0 (создатель) нажимает “🚀 Начать игру”
